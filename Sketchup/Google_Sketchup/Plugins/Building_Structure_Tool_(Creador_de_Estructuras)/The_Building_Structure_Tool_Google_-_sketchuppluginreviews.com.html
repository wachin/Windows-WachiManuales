The Building Structure Tool Google SketchUp Plugin Review

The Building Structure Tool plugin created by Tak2Hata on SketchUcation.com is a useful tool for design professionals such as structural engineers and even architects. A note that I would like to bring to your attention early in this post is that the developer is from Japan and is offering this plugin FREE though please consider donating to the 2011 Japanese Earthquake and Tsunami fund as those people are still in desperate need for aid. You can donate by:
Going to World Vision or text “4japan” to 20222 to send a $10 donation which will show up on your phone bill.
Or the American Red Cross also accepts donations via text message. Text “redcross” to 90999, and you can make a $10 donation. And finally you can contribute $10 via the Salvation Army – text “japan” or “quake” to 80888.

And now back to the review… The Structure Tool Plugin works great for those with a generous understanding of building structure design. For those that do not know the design code it may be difficult to fully understand how to use. The plugin allows you to create steel girders, beams, and columns in addition to concrete girders, beams, columns, footers and walls. And you can extend these objects and connect them with each other seamlessly with other tools available. Note that this tool is still in the Beta version so changes and updates should be coming soon.

Some elements missing are the standard (American) design specifications and even an ability to custom create your own design of beams or columns.

Overall, the Building Structure Tool plugin created by Tak2Hata on SketchUcation.com is a useful plugin for building professionals. At the time of this review this plugin is FREE and is available at SketchUcation.com. I give this plugin 4 out 5 stars.

Download Notes: Remember to be logged in with SketchUcation in order to see the file download.
