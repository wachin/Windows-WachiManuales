Align Tool Google SketchUp Plugin Review

Rating: 4.1/5 (25 votes cast)

The Align Tool plugin created by Didier Bur on SketchUcation.com is an essential tool to help you with aligning objects and components in your model. With just a simple right click and then defining a few points you can align objects precisely and quickly.

The Align Tool works really well, but it does take a few tries to get used to how it works and to utilize it to its full potential. The align tool works almost exactly like the AutoCAD align tool in the 3D view. The plugin is great and a button would make this plugin that much better but for the most part is great!

Overall, The Align Tool plugin created by Didier Bur is a fabulous plugin. At the time of this review this plugin is free and is available at SketchUcation.com. I give this plugin 5 out 5 stars!

Download Notes: Be sure to be logged into SketchUcation in order to download the file.
