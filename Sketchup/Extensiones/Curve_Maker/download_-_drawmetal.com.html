Installation

Effective March 2014, the installation process has changed.  The Curve Maker, Taper Maker and Stock Maker plugins are now distributed and installed independently of each other.  So, for example, if you only use Curve Maker, then you need only download and install that plugin.

If you have installed versions 1.3.6 or earlier of the plugins, you should DELETE THE FOLLOWING FILES from SketchUp's plugin folder prior to installing the latest version

ctsMaker (a folder - delete the folder including all of its contents)
curve_maker.rb
taper_maker.rb
stock_maker.rb

Are you a member of SketchUCation?  If you are, the plugins may be conveniently installed via the SketchUCation Plugin Store.

If you are not a member of SketchUCation, you should be, but you can get the plugins here:

Step 1.  Download any or all of the following files:

dm_curve_maker_plugin_1_4_1.rbz [v1.4.1 - Updated 2014-05-05]

dm_taper_maker_plugin_1_4.rbz [v1.4 - Updated 2014-03-20]

dm_stock_maker_plugin_1_4.rbz [v1.4 - Updated 2014-03-20]


Step 2.  In SketchUp, go to Preferences > Extensions and click the “Install Extension” button to access the downloaded files and install the plugins. (If there isn't an "Install Extension" button, then you need to upgrade to a newer version of SketchUp®)
